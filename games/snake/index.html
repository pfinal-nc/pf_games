贪吃蛇

经典的贪吃蛇游戏，控制蛇吃食物并避免碰到墙壁或自己。

得分: 0
最高分: 0

游戏规则：

使用键盘方向键（上、下、左、右）控制蛇的移动。吃到食物可以增加分数和蛇的长度。避免撞到墙壁或自己的身体，否则游戏结束。随着分数增加，蛇的移动速度会逐渐加快，难度也会增加。

控制方式：电脑使用方向键，移动设备可以使用屏幕下方的方向按钮或滑动屏幕。
